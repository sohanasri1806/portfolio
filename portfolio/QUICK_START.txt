⚡ QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

Step 1: Update Your Name
  └─ Edit index.html → Find "Your Name" → Replace with your name

Step 2: Add GitHub Username
  └─ Edit js/main.js → Find CONFIG → Update github.username

Step 3: Update Contact Info
  └─ Edit index.html → Find contact section → Update email, phone, location

Step 4: Add Your Skills
  └─ Edit data/skills.json → Add your skills in each category

Step 5: Add Your Projects
  └─ Edit data/projects.json → Add your projects

Step 6: Test Locally
  └─ Run: python -m http.server 8000
  └─ Visit: http://localhost:8000

Step 7: Deploy
  └─ See DEPLOYMENT.md for easy deployment options
