## 📞 **Need Help?**

### **Common Questions**

**Q: How do I change my name?**
A: Edit `index.html`, find "Your Name" in hero section

**Q: How do I add a project?**
A: Add entry to `data/projects.json` array

**Q: How do I change colors?**
A: Modify `:root` variables in `css/styles.css`

**Q: Can I use without customization?**
A: Yes, but replace placeholder content with yours

**Q: Is it mobile responsive?**
A: Yes! Works on all devices
