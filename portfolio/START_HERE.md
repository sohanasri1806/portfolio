## 🚀 Quick Start (5 minutes)

### **Step 1: Customize Content**

Edit `index.html`:
```html
<h1>Hi, I'm <span class="highlight">Your Name</span></h1>
```

### **Step 2: Add Your GitHub**

Edit `js/main.js`:
```javascript
github: {
    username: 'your-github-username',
    token: ''
}
```

### **Step 3: Update Data Files**

- `data/skills.json` - Your skills
- `data/projects.json` - Your projects
- `data/experience.json` - Your jobs
- `data/achievements.json` - Your awards

### **Step 4: Test Locally**

```bash
python -m http.server 8000
# Visit http://localhost:8000
```

### **Step 5: Deploy**

Choose one (see DEPLOYMENT.md for details):
- **Vercel** - `vercel` (recommended)
- **Netlify** - Drag & drop
- **GitHub Pages** - Push to repo
- **Traditional** - FTP upload

## 🎨 Customization Guide

### **Change Name**
Open `index.html`, find hero section, change "Your Name"

### **Change Colors**
Open `css/styles.css`, modify `:root` CSS variables:
```css
--primary-color: #00d4ff;      /* Cyan */
--background-color: #0a0a0a;   /* Black */
--text-color: #e0e0e0;         /* Light gray */
```

### **Change Skills**
Edit `data/skills.json` - Add/remove skill cards

### **Change Projects**
Edit `data/projects.json` - Add/remove projects

### **Change Experience**
Edit `data/experience.json` - Add/remove jobs

### **Change Achievements**
Edit `data/achievements.json` - Add/remove awards

## 💡 Pro Tips

### **For Best Results**

1. **Keep It Updated** - Add projects regularly
2. **Use Real Data** - Avoid placeholder text
3. **Quality Over Quantity** - 3 great projects > 10 mediocre ones
4. **Professional Photos** - Use good headshot if possible
5. **Test on Mobile** - Always check mobile view
6. **Get Feedback** - Ask friends to review
7. **Optimize Images** - Compress for fast loading
8. **Monitor Traffic** - Add Google Analytics

### **Content Tips**

- **About:** Keep it 2-3 paragraphs max
- **Projects:** Highlight problem solved, not just what was built
- **Skills:** Group by expertise level or domain
- **Experience:** Use action verbs, quantify achievements
- **Links:** Ensure all links work before deploying
